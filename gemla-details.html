Virtual Art Museum website

HMTL, CSS, React, MongoDB

Final project for web programming class. Worked on a team of 3 to create an ineractive photo upload website, created with React frameworks. We used mongoDB to create a functional log in page and artwork database.

Role: front-end developer  |  Date: 2022

what it is

Some features of this website include a functional log in and sign up page. When the user is not logged on, they can view artworks but cannot add or edit artworks. Once the user is logged in, they can post artworks, and edit artworks they created.
