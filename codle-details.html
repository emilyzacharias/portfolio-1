Codle

HMTL, CSS, React

Codle is a website I created for fun that works similar to Wordle, but the user does not know which letters are correct.

Role: designer, developer  Date: 2024

Coding Codle

Codle was made with HTML, CSS, and React. It features a simple, familiar UI. The website launches on a start screen that explains to the user how to play. After clicking the start button, the user is taken to a new screen with 8 slots for word guesses, as well as an onscreen keyboard. The user can use that keyboard to type their answers or keep notes of which letters they have deduced are in the final answer by toggling "notes mode."
